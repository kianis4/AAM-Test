## How Each Study Guide Is Structured

Every guide follows the same ADHD-friendly format so you know what to expect:

1. **"What Is This Document About?"** — Plain-English overview in 2-3 sentences
2. **Sections with clear headers** — Each major topic gets its own section
3. **Bold key terms** — The important stuff jumps out at you
4. **"Why This Matters" explanations** — Context for why you should care about each fact
5. **Real examples from the materials** — Not hypothetical, pulled directly from the source
6. **Tables for quick reference** — Numbers, comparisons, and lists in easy-to-scan format
7. **Practice Questions & Answers** — Short answer and scenario-based, matching the test format
